HTML

  • Hypertext Markup Language (HTML) is used to create the stucture of a webpage.
  • When we create the structure for our webpage, we used a series of HTML elements to define the content framework.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An HTML Attribute defines additional information about an element.
  • HTML Attributes are always defined in the begining element, never the closing element.
  • Elements that are nested inside other elements are called child element.
  • The element that holds all the children is knownn as the parent element.
  • In Semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • Semantic HTML elements make your HTML easier to read by other developers and the browser.
  • An element refers to the entire element from opening tag to closing tag.
  • Elements give structure to the content on the webpage.
  • Tag refers only to what appears inside the angle brackets.

CSS

  • Cascading Style Sheet (CSS) is a programming Language used by developers to define how webpage is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organization.
  • CSS can also make a website simpler to use with a polished design and consistent organization.
  • There are three ways to style a webpage using CSS: Inline CSS, Internal CSS Style Sheet, and External CSS Style Sheet.
  • When a developer wants to style an element directly in the HTML file, they use Inline CSS.
  • Internal Style Sheet contains rules for webpage in the head section of the HTML file. The rules only apply to that page.
  • The most common way to style a webpage is by using an External CSS Style Sheet. This is preferred in most cases because it allows developers to keep all their rules in a separate file, which makes design changes easier.
  • When using an External Style Sheet, we link the file using a link element placed in the HTML's head.
  • A margin indicates how much space we want arround the outside of an element.
  • A padding indicates how much space we want arround the content inside an element.

Git

  • Git & GitHub is a version control software and cloud hosting service that let us keep track of changes we make to our code.
  • To access Git use the command line.
  • Git commands always start with git and are used to save and track version of your code.
  • A branch is a copy or version of the code that can be worked on without affecting the default branch.
  • Our default branch is called "main". We should'nt do any coding directly in the "main" branch.
  • To make changes to the main branch. we should create a feature branch.
  • A feature branch is a copy of the main branch.
  • In the feature branch, we make commits of all changes we make to the code while working.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript (JS) is a high-level dynamic programming Language that allows users to interact with websites.
  • Control flow is the order in which a computer executes code in a script. In JS, the computer reads code from tor to bottom, or in a linear way.
  • The order of the code affects the outcome. JS read from top to bottom, and order counts.
  • Control flow can be interrupted, however. We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • A variable is a named container that allows us to store data in our code.
  • An array is a single variable that is used to hold a group of data.
  • A for loop uses the predictable pattern of indices to perfom a task on all items in an array by allowing a single code block to be executed over an over.
  • A function is simply a set of instructions that tells the computer how to perfom a certain task.
  • In order to use a function in JavaScript, we need to define the function and then we must call the function for the function to execute.
